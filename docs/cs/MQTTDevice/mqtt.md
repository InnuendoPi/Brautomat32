# Zařízení MQTT

Brautomat je samostatný ovladač a nemůže komunikovat s druhým Brautomat nebo CraftBeerPi4. RaspberryPi, kontejner Docker nebo jiné prostředí Python3 není vyžadováno. Brautomat se zaměřuje na jednoduché, intuitivní a cenově dostupné.\
Hardware (ESP, teplotní čidla, displej, GGM IDS2) a obvodová deska jsou v obou projektech identické. Přepínání mezi firmwarem Brautomat a MQTTDevice je možné kdykoli bez provádění jakýchkoli změn. Zařízení [MQTTDevice](https://github.com/InnuendoPi/MQTTDevice32pIO) se používá s CraftBeerPi4.
