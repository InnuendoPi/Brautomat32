# AutoTune PID

Pro Brautomat byla vyvinuta samostatná knihovna PID AutoTune InnuAPID. Výchozím bodem byla knihovna Arduino PID [Brett Beauregard](https://github.com/br3ttb/Arduino-PID-Library), stejně jako doplňky AutoTune a PIDBoil pro CraftbeerPi.

Regulace PID má za úkol generovat a udržovat konstantní teplotu rmutu bez ohledu na objem kotle nebo druh dodávky tepelné energie.
