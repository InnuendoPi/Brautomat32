# Základní nastavení

Brautomat vyžaduje alespoň jeden teplotní senzor a buď indukční varnou desku GGM IDS, nebo varnou desku ovládanou relé. V následujícím základním nastavení se konfigurují pouze požadované parametry, zatímco volitelná nastavení jsou vysvětlena později.  

Základní nastavení je vysvětleno pomocí indukční varné desky GGM IDS2. Pro reléově ovládanou varnou desku je konfigurace analogická.  

Toto základní nastavení popisuje varný systém s varnou konvicí. Nastavení dalších kotlů (např. varných kotlů nebo kotlů HLT) se řídí stejným principem.
