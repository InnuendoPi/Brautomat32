# Plán kaše

Brautomat pracuje s tabulkovým plánem rmutování. Tento plán je zpracován řádek po řádku shora dolů. Každý řádek představuje krok kaše. Vlastnosti rmutovacího kroku jsou uvedeny ve sloupcích každého řádku.

Rmutový plán také zahrnuje kroky vaření mladiny, chmelové přísady a přísady.

Začátečníkům se doporučuje, aby si před prvním dnem vaření přečetli kapitolu _Plán kaše_. Téměř všechny funkce jsou intuitivně srozumitelné. Existují však i méně zřejmé funkce.
