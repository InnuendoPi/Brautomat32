# Il caso Fregga

La FreggaCase è stata progettata dal birraio per hobby Fregga. I file di stampa 3D sono memorizzati nell'archivio Gehaeuse/FreggaCase/FreggaCase.zip.\
L'alloggiamento è diviso in due gruppi, la base dell'alloggiamento e il coperchio dell'alloggiamento. Ogni assieme ha diverse opzioni.

## Opzioni di copertura dell'alloggiamento

* con display, modulo PWM e interruttore principale
* con display, due moduli PWM e interruttore principale
*Solo visualizzazione
* senza ritagli (nessun display, nessun PWM, nessun interruttore principale)
