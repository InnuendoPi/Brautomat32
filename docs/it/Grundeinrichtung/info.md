# Configurazione di base

Il Brautomat richiede almeno un sensore di temperatura e un piano cottura a induzione GGM IDS o un piano cottura controllato da relè. Nelle seguenti impostazioni di base vengono configurati solo i parametri richiesti, mentre le impostazioni opzionali vengono spiegate più avanti.  

L'impostazione base viene spiegata utilizzando il piano cottura a induzione GGM IDS2. Per un piano cottura comandato da relè la configurazione è analoga.  

Questa impostazione di base descrive un sistema di erogazione con un bollitore per il mosto. L'impostazione di caldaie aggiuntive (ad esempio caldaie per la preparazione della birra o caldaie HLT) segue lo stesso principio.
