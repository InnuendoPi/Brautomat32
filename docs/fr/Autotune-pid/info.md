# AutoTune PID

Une bibliothèque PID AutoTune distincte, InnuAPID, a été développée pour le Brautomat. Le point de départ était la bibliothèque Arduino PID [Brett Beauregard](https://github.com/br3ttb/Arduino-PID-Library), ainsi que les addons AutoTune et PIDBoil pour CraftbeerPi.

La régulation PID a pour mission de générer et de maintenir une température de purée constante, quel que soit le volume de la chaudière ou le type d'apport d'énergie thermique.
