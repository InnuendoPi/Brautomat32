# Fonctions des boutons

## Description de tous les boutons

Le plan de brassage comporte des boutons pour modifier, déplacer ou supprimer une ligne.

### Le bouton Enregistrer du plan de purée

Le bouton Enregistrer dans l'en-tête du tableau du plan de brassage enregistre le contenu du tableau dans un fichier (JSON).

### Le bouton Mettre à jour le plan de purée (bouton Recharger)

Cette fonction relit la table du fichier. Veuillez noter que toutes les modifications non enregistrées (bouton vert) seront ignorées sans requête. Ce bouton a la fonction de sortie du mode éditeur comme deuxième fonction.

### Le bouton de sortie de l'éditeur

Cette fonction quitte le mode éditeur sans enregistrer les modifications dans le plan de brassage.

### Le bouton Supprimer

L'utilisation du bouton Supprimer supprime la table entière. Veuillez noter que les modifications ne sont appliquées que lorsque vous cliquez sur Enregistrer le tableau (bouton vert).

### Le bouton +

Une nouvelle ligne est ajoutée à l'aide du bouton plus. Veuillez noter que la nouvelle ligne du tableau doit être acceptée en cliquant sur le symbole de sauvegarde bleu dans la ligne du tableau et enfin le tableau est enregistré en cliquant sur le symbole de sauvegarde vert.

### Le bouton graphique

Le bouton graphique peut être utilisé pour afficher ou masquer la représentation visuelle de la courbe de température. De plus, des graphiques individuels peuvent être affichés ou masqués en cliquant sur le nom du graphique. La valeur par défaut est définie sur visible.

*Astuce : Si la courbe de température n'est pas intéressante, cliquez simplement sur le bouton graphique vert. Dès que le graphique est masqué, ouvrez une fois la configuration du système et cliquez sur Enregistrer. C'est ainsi que le paramètre Masquer ou afficher l'historique graphique des températures est enregistré.*

### Le bouton Paramètres du plan de purée (bouton Base de données)

Dans les paramètres du plan de brassage, outre le nom du plan de brassage et le contrôle du temps, la modification, l'importation, l'exportation et la suppression des plans de brassage sont également disponibles.

### Le bouton Afficher le plan de purée (bouton Réduire)

Le bouton Afficher le plan de brassage a une fonction de réduction : pendant le brassage, le plan de brassage utilise une quantité inutile d'espace à l'écran. Avec la réduction, vous pouvez réduire et développer la table du plan de brassage.

### Le bouton Afficher le graphique (bouton Réduire)

Le bouton Afficher le graphique a une fonction de réduction : le graphique utilise une quantité inutile d'espace à l'écran pendant le brassage. Avec la réduction, vous pouvez réduire et agrandir le graphique. Le graphique est également mis à jour davantage lorsqu'il est réduit.

### Le bouton graphique

Le bouton graphique peut être utilisé pour afficher et masquer la représentation visuelle de la courbe de température. De plus, des graphiques individuels peuvent être affichés ou masqués en cliquant sur le nom du graphique. La valeur par défaut est visible.

*Astuce : Si la courbe de température ne vous intéresse pas, cliquez simplement sur le bouton graphique vert. Dès que le graphique est masqué, ouvrez une fois la configuration du système et cliquez sur Enregistrer. Cela enregistre le paramètre Masquer ou afficher l’historique graphique de la température*.

### Paramètres du bouton du plan de purée (bouton de la base de données)

Dans les paramètres du plan de brassage, en plus du nom du plan de brassage et du contrôle du temps, les modifications du plan de brassage, l'importation, l'exportation et la suppression de plans de brassage sont également disponibles.

### Le bouton Afficher le plan de purée (bouton Réduire)

Le bouton Afficher le plan de brassage a une fonction pliable : pendant le brassage, le plan de brassage occupe inutilement de l'espace sur l'écran. Avec la réduction, la table du plan de purée peut être réduite et agrandie.

### Le bouton Afficher le diagramme (bouton Réduire)

Le bouton Afficher le graphique a une fonction de réduction : pendant la préparation, le graphique occupe une quantité inutile d'espace à l'écran. Avec la réduction, le graphique peut être réduit et développé. Les graphiques sont également inclusmis à jour davantage à l’état plié.
