# Configuration de base

Le Brautomat nécessite au moins une sonde de température et soit une table de cuisson à induction GGM IDS, soit une table de cuisson commandée par relais. Dans le réglage de base suivant, seuls les paramètres requis sont configurés, tandis que les paramètres facultatifs sont expliqués plus tard.  

Le réglage de base est expliqué à l'aide de la table à induction GGM IDS2. Pour une table de cuisson commandée par relais, la configuration est analogue.  

Ce réglage de base décrit un système de brassage avec une cuve de brassage de purée. La mise en place de chaudières supplémentaires (par exemple chaudières de brassage ou HLT) suit le même principe.
