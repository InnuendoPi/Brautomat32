# Le Dashboard

![Dashboard]([[CHEMIN_0]])

Le Dashboard est utilisé en brassage. Dès que les tables de cuisson, les actionneurs et les capteurs sont configurés, le Dashboard regroupe toutes les informations importantes en un seul endroit.

## Activez Dashboard.

Le Dashboard est activé dans les paramètres système :

![Dashboard]([[CHEMIN_1]])

En fonction de la configuration existante, les chaudières, les actionneurs et les capteurs peuvent être affichés dans Dashboard. La taille de l'interface Web de Brautomat peut être réduite à l'aide de la sélection affichée dans l'image :

* Les éléments de configuration des tables de cuisson, des actionneurs et des capteurs sont désactivés
* Les éléments de plaques de cuisson, actionneurs et capteurs sont activés dans Dashboard.

## Utilisez Dashboard.

Si le Dashboard est activé, le Brautomat change automatiquement son apparence lorsqu'un processus de brassage est démarré :

* le plan de brassage est automatiquement réduit
* L'étape de purée active est affichée au-dessus du controller Deck.
* L'étape suivante du brassage est affichée au-dessus du graphique de la courbe de température.
* l'étape suivante du brassage est affichée avec un symbole rouge ou vert selon autonext.

Tous les éléments de contrôle des tables de cuisson et des actionneurs restent dans le Dashboard. Le plan de brassage peut être rouvert à tout moment à l'aide du bouton de réduction. Si vous cliquez sur le bouton pause, le plan de brassage s'ouvre automatiquement. Les éléments masqués peuvent également être réaffichés pendant un processus de préparation via les paramètres du système.
