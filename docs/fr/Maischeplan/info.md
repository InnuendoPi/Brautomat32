# Le plan purée

Le Brautomat fonctionne avec un plan de brassage tabulaire. Ce plan est traité ligne par ligne de haut en bas. Chaque ligne représente une étape de purée. Les propriétés d'une étape de purée sont affichées dans les colonnes de chaque ligne.

Le plan de purée comprend également des étapes d’ébullition du moût, des ajouts de houblon et des additifs.

Il est recommandé aux débutants de lire le chapitre _Mash Plan_ avant le premier jour de brassage. Presque toutes les fonctions sont intuitivement compréhensibles. Mais il existe aussi des fonctions moins évidentes.
