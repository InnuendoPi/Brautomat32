# MQTTAppareil

Le Brautomat est un contrôleur autonome et ne peut pas communiquer avec un deuxième Brautomat ou CraftBeerPi4. Un RaspberryPi, un conteneur Docker ou un autre environnement Python3 n'est pas requis. Le Brautomat se concentre sur la simplicité, l'intuitivité et le prix abordable.\
Le matériel (ESP, capteurs de température, affichage, GGM IDS2) et le circuit imprimé sont identiques dans les deux projets. Un basculement entre les firmwares Brautomat et MQTTDevice est possible à tout moment sans apporter de modifications. Le [MQTTDevice](https://github.com/InnuendoPi/MQTTDevice32pIO) est utilisé avec CraftBeerPi4.
