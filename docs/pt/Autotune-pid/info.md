# AutoTune PID

Uma biblioteca separada PID AutoTune InnuAPID foi desenvolvida para o Brautomat. O ponto de partida foi a biblioteca PID do Arduino [Brett Beauregard](https://github.com/br3ttb/Arduino-PID-Library), bem como os addons AutoTune e PIDBoil para CraftbeerPi.

O controle PID tem a função de gerar e manter uma temperatura de mostura constante, independente do volume da caldeira ou do tipo de fornecimento de energia térmica.
