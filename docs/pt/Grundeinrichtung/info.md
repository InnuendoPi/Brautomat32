#Configuração básica

O Brautomat requer pelo menos um sensor de temperatura e uma placa de indução GGM IDS ou uma placa controlada por relé. Na configuração básica a seguir, apenas os parâmetros necessários são configurados, enquanto as configurações opcionais são explicadas posteriormente.  

A configuração básica é explicada utilizando a placa de indução GGM IDS2. Para uma placa controlada por relé, a configuração é análoga.  

Esta configuração básica descreve um sistema de preparo com uma chaleira para preparo de purê. A instalação de caldeiras adicionais (por exemplo, caldeiras de cerveja ou HLT) segue o mesmo princípio.
