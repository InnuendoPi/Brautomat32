# O plano de purê

O Brautomat funciona com um plano de mosturação tabular. Este plano é processado linha por linha, de cima para baixo. Cada linha representa uma etapa de mosturação. As propriedades de uma etapa de mosturação são mostradas nas colunas de cada linha.

O plano de mosturação também inclui etapas de fervura do mosto, adições e aditivos de lúpulo.

Recomenda-se que os iniciantes leiam o capítulo _Mash Plan_ antes do primeiro dia de preparação. Quase todas as funções são intuitivamente compreensíveis. Mas também existem funções menos óbvias.
