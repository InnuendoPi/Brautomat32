# O Dashboard

![Dashboard](/docs/img/dashboard160.jpg)

O Dashboard é usado na fabricação de cerveja. Assim que placas, atuadores e sensores são configurados, o Dashboard combina todas as informações importantes em um só lugar.

## Ative Dashboard.

O Dashboard está ativado nas configurações do sistema:

![Dashboard](/docs/img/dashboard160-1.jpg)

Dependendo da configuração existente, caldeiras, atuadores e sensores podem ser exibidos em Dashboard. A interface web de Brautomat pode ser reduzida em tamanho usando a seleção mostrada na imagem:

* Os elementos de configuração das placas, atuadores e sensores estão desativados
* Os elementos de placas, atuadores e sensores são ativados em Dashboard.

## Use Dashboard.

Se o Dashboard estiver ativado, o Brautomat muda automaticamente sua aparência quando um processo de preparação é iniciado:

* o plano mash é automaticamente recolhido
* A etapa de mosturação ativa é exibida acima de controller Deck.
* O próximo passo de mosturação é exibido acima do gráfico da curva de temperatura
* o próximo passo de mosturação é exibido com um símbolo vermelho ou verde dependendo de autonext.

Todos os elementos de controle de placas e atuadores permanecem no Dashboard. O plano mash pode ser aberto novamente a qualquer momento usando o botão recolher. Se o botão de pausa for clicado, o plano de mosturação será aberto automaticamente. Os elementos ocultos também podem ser exibidos novamente durante o processo de preparo através das configurações do sistema.
