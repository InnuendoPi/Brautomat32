# O quadro do projeto

A placa do projeto Brautomat facilita muito a construção e operação. O GGM IDS, os sensores, os atuadores e o display são conectados aos terminais de parafuso da Brautomat.

A placa é uma placa breakout para os microcontroladores ESP8266 e ESP32. Sensores, interruptores, etc. podem ser conectados aos blocos de terminais de parafuso. Ambas as placas podem ser utilizadas universalmente e não estão limitadas aos projetos Braautomat ou MQTTDevice. Duas conexões são fixas:

* GPIO D3 com resistor 4k7 destina-se a sensores Dallas DS18B20.
* GPIO D8 está conectado à conexão de uma campainha piezoelétrica na placa
