# The FreggaCase

FreggaCase was designed by hobby brewer Fregga.
3D print files are in `Gehaeuse/FreggaCase/FreggaCase.zip`.

## Lid options

* display + one PWM + main switch
* display + two PWM + main switch
* display only
* no cutouts (blank lid)
