# Safety check before first heat test

This is a mandatory pre-check before the first powered heating run.
Run the first test with water only and under supervision.

## Pre-start checklist

- Setup is stable, dry, and clear of flammable materials.
- Mains power is protected appropriately (for example RCD/FI).
- Enclosure, connectors, and cables are undamaged.
- Sensor assignment is correct and readings are plausible.
- Emergency stop/shutdown path is clear (plug/switch reachable).
- Browser connection is active, but remember: process continues if client disconnects.

## First heat test (water)

1. Fill kettle with water (typical partial volume).
2. Set a moderate target (for example 55 to 60 °C).
3. Start heating and monitor temperature rise.
4. Verify target is reached and held stably.
5. Test manual stop behavior.

## Abort immediately if

- unusual smell or heating at connectors/cables
- implausible temperature jumps
- unexpected continuous heating or missing shutdown behavior
