# AutoTune PID

Dla Brautomat opracowano osobną bibliotekę PID AutoTune InnuAPID. Punktem wyjścia była biblioteka Arduino PID [Brett Beauregard](https://github.com/br3ttb/Arduino-PID-Library), a także dodatki AutoTune i PIDBoil dla CraftbeerPi.

Sterowanie PID ma za zadanie wytworzenie i utrzymanie stałej temperatury zacieru, niezależnie od objętości kotła czy rodzaju dostawy energii cieplnej.
