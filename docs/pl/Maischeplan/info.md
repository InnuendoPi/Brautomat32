# Plan zacieru

Brautomat działa z tabelarycznym planem zacierania. Plan ten jest przetwarzany linia po linii od góry do dołu. Każda linia reprezentuje krok zacierania. Właściwości etapu zacierania są pokazane w kolumnach każdego wiersza.

Plan zacieru obejmuje również etapy gotowania brzeczki, dodatki chmielowe i dodatki.

Początkującym zaleca się przeczytanie rozdziału _Plan zacieru_ przed pierwszym dniem warzenia. Prawie wszystkie funkcje są intuicyjnie zrozumiałe. Ale są też mniej oczywiste funkcje.
