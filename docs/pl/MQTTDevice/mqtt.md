# Urządzenie MQTT

Brautomat jest samodzielnym kontrolerem i nie może komunikować się z drugim Brautomat ani CraftBeerPi4. RaspberryPi, kontener Docker lub inne środowisko Python3 nie jest wymagane. Brautomat skupia się na prostocie, intuicyjności i przystępnej cenie.\
Sprzęt (ESP, czujniki temperatury, wyświetlacz, GGM IDS2) i płytka drukowana są identyczne w obu projektach. Przełączanie pomiędzy oprogramowaniem Brautomat a MQTTDevice jest możliwe w dowolnym momencie bez dokonywania jakichkolwiek zmian. [MQTTDevice](https://github.com/InnuendoPi/MQTTDevice32pIO) jest używane z CraftBeerPi4.
