# AutoTune PID

Ett separat PID AutoTune bibliotek InnuAPID utvecklades för Brautomat. Utgångspunkten var Arduino PID-biblioteket [Brett Beauregard](https://github.com/br3ttb/Arduino-PID-Library), såväl som tilläggen AutoTune och PIDBoil för CraftbeerPi.

Reglaget PID har till uppgift att generera och upprätthålla en konstant mäsktemperatur, oavsett pannvolym eller typ av värmeenergitillförsel.
