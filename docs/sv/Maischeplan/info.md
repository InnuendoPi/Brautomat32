# Mäskplanen

Brautomat fungerar med en tabellform för mäskning. Denna plan bearbetas rad för rad från topp till botten. Varje linje representerar ett mäsksteg. Egenskaperna för ett mäsksteg visas i kolumnerna i varje rad.

I mäskplanen ingår även vörtkokningssteg, humletillsatser och tillsatser.

Det rekommenderas att nybörjare läser kapitlet _Mash Plan_ innan den första bryggdagen. Nästan alla funktioner är intuitivt begripliga. Men det finns också mindre uppenbara funktioner.
