# Dashboard

![Dashboard](/docs/img/dashboard160.jpg)

Dashboard används vid bryggning. Så snart hällar, ställdon och sensorer är konfigurerade, kombinerar Dashboard all viktig information på ett ställe.

## Aktivera Dashboard.

Dashboard aktiveras i systeminställningarna:

![Dashboard](/docs/img/dashboard160-1.jpg)

Beroende på befintlig konfiguration kan pannor, ställdon och givare visas i Dashboard. Webbgränssnittet för Brautomat kan minskas i storlek med hjälp av urvalet som visas i bilden:

* Elementen för konfigurering av hällar, ställdon och sensorer är avaktiverade
* Elementen för hällar, ställdon och sensorer aktiveras i Dashboard.

## Använd Dashboard.

Om Dashboard är aktiverad ändrar Brautomat automatiskt utseende när en bryggprocess startas:

* mäskplanen komprimeras automatiskt
* Det aktiva mäsksteget visas ovanför controller Deck.
* Nästa mäsksteg visas ovanför temperaturkurvans grafik
* nästa mäsksteg visas med en röd eller grön symbol beroende på autonext.

Alla element för styrning av hällar och ställdon finns kvar i Dashboard. Mashplanen kan när som helst öppnas igen med komprimeringsknappen. Om pausknappen klickas öppnas mäskplanen automatiskt. Dolda element kan också visas igen under en bryggprocess via systeminställningarna.
