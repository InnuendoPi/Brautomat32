# De Dashboard

![Dashboard](/docs/img/dashboard160.jpg)

De Dashboard wordt gebruikt bij het brouwen. Zodra kookplaten, actuatoren en sensoren zijn geconfigureerd, combineert de Dashboard alle belangrijke informatie op één plek.

## Activeer Dashboard.

De Dashboard wordt geactiveerd in de systeeminstellingen:

![Dashboard](/docs/img/dashboard160-1.jpg)

Afhankelijk van de bestaande configuratie kunnen ketels, actoren en sensoren worden weergegeven in Dashboard. De webinterface van Brautomat kan worden verkleind met behulp van de selectie die in de afbeelding wordt weergegeven:

* De elementen voor het configureren van kookplaten, actuatoren en sensoren zijn gedeactiveerd
* De elementen van kookplaten, actuatoren en sensoren worden geactiveerd in Dashboard.

## Gebruik Dashboard.

Als de Dashboard is geactiveerd, verandert de Brautomat automatisch van uiterlijk wanneer een brouwproces wordt gestart:

* het maischplan wordt automatisch ingeklapt
* De actieve mash-stap wordt weergegeven boven de controller Deck.
* De volgende maischstap wordt weergegeven boven de temperatuurcurvegrafiek
* de volgende maischstap wordt weergegeven met een rood of groen symbool, afhankelijk van autonext.

Alle elementen voor het aansturen van kookplaten en actuatoren blijven in de Dashboard. Het maischplan kan op elk moment weer worden geopend via de knop Samenvouwen. Als er op de pauzeknop wordt geklikt, wordt het maischplan automatisch geopend. Ook tijdens een brouwproces kunnen verborgen elementen via de systeeminstellingen opnieuw worden getoond.
