# Dashboard

![Dashboard](/docs/img/dashboard160.jpg)

Dashboard brukes i brygging. Så snart koketopper, aktuatorer og sensorer er konfigurert, kombinerer Dashboard all viktig informasjon på ett sted.

## Aktiver Dashboard.

Dashboard er aktivert i systeminnstillingene:

![Dashboard](/docs/img/dashboard160-1.jpg)

Avhengig av eksisterende konfigurasjon kan kjeler, aktuatorer og sensorer vises i Dashboard. Nettgrensesnittet til Brautomat kan reduseres i størrelse ved å bruke utvalget vist på bildet:

* Elementene for konfigurering av koketopper, aktuatorer og sensorer er deaktivert
* Elementene til kokeplater, aktuatorer og sensorer aktiveres i Dashboard.

## Bruk Dashboard.

Hvis Dashboard er aktivert, endrer Brautomat automatisk utseende når en bryggeprosess startes:

* mash-planen er automatisk kollapset
* Det aktive mash-trinnet vises over controller Deck.
* Neste mesketrinn vises over temperaturkurvegrafikken
* neste mash-trinn vises med et rødt eller grønt symbol avhengig av autonext.

Alle elementer for styring av koketopper og aktuatorer forblir i Dashboard. Mash-planen kan åpnes igjen når som helst ved å bruke komprimeringsknappen. Hvis du klikker på pauseknappen, åpnes mash-planen automatisk. Skjulte elementer kan også vises igjen under en bryggeprosess via systeminnstillingene.
