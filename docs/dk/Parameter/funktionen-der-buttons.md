# Button functions

## Description of all buttons

The mash plan has buttons to edit, move or delete a line.

### The save button of the mash plan

The save button in the header of the mash plan table saves the contents of the table in a file (JSON).

### The Update Mash Plan button (Reload Button)

This function re-reads the table from the file. Please note that all unsaved changes (green button) will be discarded without query. This button has the exit editor mode function as a second function.

### The editor's exit button

This function exits editor mode without saving the changes in the mash plan.

### The delete button

Using the delete button deletes the entire table. Please note that the changes are only applied when you click on Save table (green button).

### The + button

A new line is added using the plus button. Please note that the new row in the table must be accepted by clicking on the blue save symbol in the table row and finally the table is saved by clicking on the green save symbol.

### The Graph Button

The graph button can be used to show or hide the visual representation of the temperature curve. Additionally, individual graphs can be shown or hidden by clicking on the graph name. The default is set to visible.

*Tip: If the temperature curve is not interesting, just click on the green graph button. As soon as the graphic is hidden, open the system configuration once and click Save. This is how the Hide or show graphic temperature history setting is saved.*

### The Settings Mash Plan Button (Database Button)

In the settings for the mash plan, in addition to the mash plan name and time control, the mash plan change, import, export and deletion of mash plans are also available.

### The Show Mash Plan Button (Collapse Button)

The Show mash plan button has a collapse function: during brewing, the mash plan uses an unnecessary amount of screen space. With collapse you can collapse and expand the mash plan table.

### The Show Chart Button (Collapse Button)

The Show Chart button has a collapse function: the graphic uses an unnecessary amount of screen space while brewing. With collapse you can collapse and expand the graphic. The graphic is also updated further when collapsed.

The graph button can be used to show and hide the visual representation of the temperature curve. In addition, individual graphics can be shown or hidden by clicking on the graphic name. The default is visible.

### The Show Mash Plan button (Collapse Button)

The Show mash plan button has a folding function: During brewing, the mash plan takes up an unnecessary amount of screen space. With collapse the mash plan table can be collapsed and expanded.

The Show Chart button has a collapse function: During brewing, the graphic takes up an unnecessary amount of screen space. With collapse the graphic can be collapsed and expanded. The graphics are also includedfurther updated in folded state.
