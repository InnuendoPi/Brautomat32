# The FreggaCase

The FreggaCase was designed by hobby brewer Fregga. The 3D print files are stored in the archive Gehaeuse/FreggaCase/FreggaCase.zip.\
The housing is divided into two assemblies, the housing base and the housing cover. Each assembly has different options.

## Housing cover options

* with display, PWM module and main switch
* with display, two PWM modules and main switch
* Display only
* without cutouts (no display, no PWM, no main switch)
