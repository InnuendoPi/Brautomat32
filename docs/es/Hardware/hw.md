## Ejemplos de actores

### Bombas

![Actores](/docs/img/Pumpe1.jpg)

Una bomba es de gran ayuda a la hora de clarificarlo o transportarlo al tanque de fermentación. La bomba utilizada debe ser apta para alimentos y adecuada para temperaturas de hasta 100 °C. Las minibombas de Rotekt son ideales para su uso en la cocina de una cervecería:

### Relé de estado sólido SSR

Con Brautomat los SSR se pueden cambiar muy fácilmente. Los relés SSR o de estado sólido encienden o apagan el circuito de 220 VCA con 3-5 VCC desde Brautomat. Los Fotek SSR son ideales para el control de temperatura y el control a través de Brautomat.

![fotek](/docs/img/fotek.jpg)

| Nombre del artículo | Enlace al artículo |
| -------------- | --------------------------------- |
| Fotek SSR 40DA | [Amazonas].

Ejemplos de aplicación para SSR: Si se enciende 220 V en un SSR Fotek a través de Brautomat, se puede conectar un enchufe para cualquier dispositivo con un enchufe al "otro extremo" de la línea de 220 V. Por ejemplo para la alimentación del agitador. O para una cocina recargable. O para un elemento calefactor anular. La fuente de alimentación de la bomba Rotek también se puede encender y apagar mediante un SSR.

### Módulo de relé

### Elementos calefactores anulares

Un elemento calefactor de anillo es útil al hervir el mosto si el rendimiento del GGM IDS ya no es suficiente o es suficiente para hervir el mosto hasta que burbujee. Puede obtener una buena selección de elementos calefactores anulares en [Douglas de Crafthardware](https://www.crafthardware.de/products/ringheizelement-3-5-kw-fuer-45-cm-kessel).
