# AutoTune PID

Se desarrolló una biblioteca PID AutoTune independiente, InnuAPID, para Brautomat. El punto de partida fue la biblioteca Arduino PID [Brett Beauregard](https://github.com/br3ttb/Arduino-PID-Library), así como los complementos AutoTune y PIDBoil para CraftbeerPi.

El control PID tiene la tarea de generar y mantener una temperatura de macerado constante, independientemente del volumen de la caldera o del tipo de suministro de energía térmica.
