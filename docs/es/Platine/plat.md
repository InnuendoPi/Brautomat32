# El tablero del proyecto.

El tablero del proyecto Brautomat facilita mucho la construcción y la operación. El GGM IDS, los sensores, los actuadores y la pantalla están conectados a los bloques de terminales de tornillo del Brautomat.

La placa es una placa de conexión para los microcontroladores ESP8266 y ESP32. A los bloques de terminales de tornillos se pueden conectar sensores, interruptores, etc. Ambas placas se pueden utilizar universalmente y no se limitan a los proyectos Braautomat o MQTTDevice. Se fijan dos conexiones:

* GPIO D3 con una resistencia de 4k7 está diseñado para sensores Dallas DS18B20.
* GPIO D8 está conectado a la conexión para un zumbador piezoeléctrico en la placa
