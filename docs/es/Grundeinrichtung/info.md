# Configuración básica

El Brautomat requiere al menos un sensor de temperatura y una placa de inducción GGM IDS o una placa controlada por relé. En la siguiente configuración básica, solo se configuran los parámetros requeridos, mientras que las configuraciones opcionales se explican más adelante.  

Se explica el ajuste básico utilizando la placa de inducción GGM IDS2. Para una placa de cocción controlada por relé, la configuración es análoga.  

Este ajuste básico describe un sistema de preparación con caldera de preparación de puré. La instalación de calderas adicionales (por ejemplo, calderas de elaboración de cerveza o HLT) sigue el mismo principio.
