# Dispositivo MQTT

El Brautomat es un controlador independiente y no puede comunicarse con un segundo Brautomat o CraftBeerPi4. No se requiere una RaspberryPi, un contenedor Docker u otro entorno Python3. El Brautomat se centra en lo simple, intuitivo y asequible.\
El hardware (ESP, sensores de temperatura, display, GGM IDS2) y la placa de circuito son idénticos en ambos proyectos. Es posible cambiar entre los firmwares Brautomat y MQTTDevice en cualquier momento sin realizar ningún cambio. El [MQTTDevice](https://github.com/InnuendoPi/MQTTDevice32pIO) se utiliza con CraftBeerPi4.
