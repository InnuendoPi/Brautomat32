# El Dashboard

![Dashboard](/docs/img/dashboard160.jpg)

El Dashboard se utiliza en la elaboración de cerveza. Tan pronto como se configuran las placas de cocción, los actuadores y los sensores, Dashboard combina toda la información importante en un solo lugar.

## Activar Dashboard.

El Dashboard se activa en la configuración del sistema:

![Dashboard](/docs/img/dashboard160-1.jpg)

Dependiendo de la configuración existente, las calderas, actuadores y sensores se pueden mostrar en Dashboard. La interfaz web de Brautomat se puede reducir de tamaño usando la selección que se muestra en la imagen:

* Los elementos de configuración de placas, actuadores y sensores están desactivados
* Los elementos de placas de cocción, actuadores y sensores se activan en Dashboard.

## Utilice Dashboard.

Si Dashboard está activado, Brautomat cambia automáticamente su apariencia cuando se inicia un proceso de preparación:

* el plan de mezcla se contrae automáticamente
* El paso de puré activo se muestra encima de controller Deck.
* El siguiente paso de macerado se muestra encima del gráfico de la curva de temperatura.
* el siguiente paso de puré se muestra con un símbolo rojo o verde dependiendo de autonext.

Todos los elementos para controlar las placas de cocción y los actuadores permanecen en el Dashboard. El plan de maceración se puede volver a abrir en cualquier momento usando el botón contraer. Si se hace clic en el botón de pausa, el plan de mezcla se abre automáticamente. Los elementos ocultos también se pueden volver a mostrar durante el proceso de elaboración a través de los ajustes del sistema.
