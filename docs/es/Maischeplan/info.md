# El plan de puré

El Brautomat funciona con un plan de maceración tabular. Este plan se procesa línea por línea de arriba a abajo. Cada línea representa un paso de puré. Las propiedades de un paso de puré se muestran en las columnas de cada fila.

El plan de macerado también incluye pasos de ebullición del mosto, adiciones de lúpulo y aditivos.

Se recomienda que los principiantes lean el capítulo _Mash Plan_ antes del primer día de preparación. Casi todas las funciones son intuitivamente comprensibles. Pero también hay funciones menos obvias.
